JavaScript (JS)

Javascript (JS) is a scripting languages, primarily used on the Web. It is used to enhance HTML pages and is commonly found embedded in HTML code. JavaScript is an interpreted language.

Operators

+, -, *, /, and % are called operators. An operator is a special character (or characters) that indicates an action to be performed.

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable naming conventions

Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.; variables are case sensitve (myNumber is different than myNUMBER); use clear names that describe the value being stored like myNumber - avoid vague letters or initials.

Functions

A function is a block of code that performs an action and returns a result; optionally takes arguments

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

An argument is what you pass into a function or method

Parameters

A parameter is a variable that's assigned to the argument. Above, number1 and number2 are parameters, and 1 and 2 are arguments.

Return

A is The return value is the method's response.

Chaining Methods

Calling a method directly on the return value of another method.

Strings

A JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

Booleans

true and false are booleans. They are not strings - they simply represent being true or false.

Undefined

An object (such as a variable) without a defined value.

NaN

Not a number.

Escape

An escape character enables you to output characters you wouldn't normally be able to, usually because the browser will interpret it differently to what you intended. In JavaScript, the backslash ( \ ) is an escape character.

Alert

The alert() method displays an alert box with a specified message and an OK button.

Comments

A comment is a programmer-readable explanation or annotation in the suorce code of a computer program. They are added with the purpose of making the source code easier for humans to understand, and are generally ignored by compilers and interpreters.

jQuery

jQuery is a JavaScript library that makes it easy to make web pages interactive.

Atrributes

An attribute is a changeable property or characteristic of some component of a program that can be set to different values. In the Hypertext Markup Language (HTML), an attribute is a characteristic of a page element, such as a font. They are specified in the start tag and usually come in name/value pairs like name="value".
